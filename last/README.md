# 员工信息管理系统

一个基于 Vue 3 + Element Plus 的员工信息管理系统，包含完整的员工管理、请假管理和工资管理功能。

## 功能特性

### 员工管理模块
- ✅ 员工信息管理（增删改查）
- ✅ 搜索功能（支持姓名、工号、部门搜索）
- ✅ 分页功能（可自定义每页显示数量）
- ✅ 表单验证

### 员工请假模块
- ✅ 请假记录管理（增删改查）
- ✅ 请假审批功能（批准/拒绝）
- ✅ 多种请假类型（年假、病假、事假、调休）
- ✅ 请假天数自动计算
- ✅ 状态筛选和类型筛选

### 员工工资模块
- ✅ 工资记录管理（增删改查）
- ✅ 工资统计（总额、平均工资、奖金总额）
- ✅ 按月筛选工资记录
- ✅ 工资组成明细（基本工资、绩效、津贴、奖金、扣除）
- ✅ 实发工资自动计算

### 通用功能
- ✅ 侧边栏导航（可折叠）
- ✅ 数据持久化（使用 localStorage 保存数据）
- ✅ 响应式设计（支持移动端）
- ✅ 现代化UI设计（Element Plus）
- ✅ 路由管理

## 技术栈

- Vue 3 (Composition API)
- Vue Router
- Element Plus
- Vite

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 `http://localhost:3000`

### 构建生产版本

```bash
npm run build
```

## 使用说明

### 员工管理
1. **查看员工列表**：系统默认加载15条示例数据
2. **添加员工**：点击右上角"添加员工"按钮，填写信息后保存
3. **编辑员工**：点击员工记录右侧的"编辑"按钮
4. **删除员工**：点击员工记录右侧的"删除"按钮
5. **搜索员工**：在搜索框输入关键词进行搜索
6. **分页浏览**：使用底部分页控件浏览不同页面的数据

### 员工请假
1. **查看请假记录**：在左侧菜单点击"员工请假"
2. **添加请假记录**：点击"添加请假记录"按钮
3. **审批请假**：对于待审批的请假，可以点击"批准"或"拒绝"
4. **筛选记录**：可以按状态、类型、员工姓名进行筛选

### 员工工资
1. **查看工资记录**：在左侧菜单点击"员工工资"
2. **查看统计信息**：顶部卡片显示工资统计数据
3. **按月筛选**：选择月份查看该月的工资记录
4. **添加工资记录**：点击"添加工资记录"按钮
5. **编辑工资**：可以编辑各项工资组成，实发工资会自动计算

## 数据存储

所有数据保存在浏览器的 localStorage 中，刷新页面后数据不会丢失。
